# 📱 ADFix Repair Manager

[![Laravel](https://img.shields.io/badge/Framework-Laravel-red.svg)](https://laravel.com)
[![TailwindCSS](https://img.shields.io/badge/CSS-Tailwind-38B2AC.svg)](https://tailwindcss.com/)

**ADFix Repair Manager** es una solución web integral diseñada para optimizar la gestión de servicios técnicos de telefonía móvil. Este sistema permite centralizar el control de reparaciones, clientes y finanzas en una interfaz intuitiva y eficiente.

🚀 **Prueba la Demo:** [ADFix](https://adfix-repair-manager.onrender.com)
- **Usuario:** `test@example.com`
- **Contraseña:** `password`

---

## ✨ Características Principales

- **Gestión de Reparaciones:** Control total del ciclo de vida de cada equipo, desde el ingreso hasta la entrega.
- **Estados Dinámicos:** Seguimiento en tiempo real del progreso (Pendiente, En Reparación, Listo, Entregado).
- **Base de Datos de Clientes:** Registro detallado de historial por cliente.
- **Módulo Financiero:** Visualización clara de ganancias, costos de repuestos y balances netos.
- **Dashboard Estadístico:** Resúmenes mensuales automáticos para la toma de decisiones basada en datos.
- **Diseño Responsive:** Acceso desde cualquier dispositivo gracias a Tailwind CSS.

## 🛠️ Stack Tecnológico

- **Backend:** [Laravel 10+](https://laravel.com/)
- **Frontend:** Blade Templates & [Tailwind CSS](https://tailwindcss.com/)
- **Base de Datos:** SQLite
- **Testing:** PHPUnit
